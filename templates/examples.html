Fake Twilio API Examples

Messages

  • Retrieve a list of messages sent

    curl {{data.Account.Sid}}:{{data.Account.AuthToken}}@{{data.host}}/2010-04-01/Accounts/{{data.Account.Sid}}/SMS/Messages.json

    • Incoming Phone Numbers

  • Requisition an incoming number with Area Code(301) and assign a Friendly Name

    curl -d "AreaCode=301&FriendlyName=Phone Number Example" --user {{data.Account.Sid}}:{{data.Account.AuthToken}} {{data.host}}/2010-04-01/Accounts/{{data.Account.Sid}}/IncomingPhoneNumbers.json

  • Get an incoming phone numbers information

    curl --user {{data.Account.Sid}}:{{data.Account.AuthToken}} {{data.host}}/2010-04-01/Accounts/{{data.Account.Sid}}/IncomingPhoneNumbers/*SID*

  • Update an incoming phone number's VoiceUrl and FriendlyName

    curl -d "FriendlyName=TVSHOW&VoiceUrl=http://www.cnn.com" --user {{data.Account.Sid}}:{{data.Account.AuthToken}} {{data.host}}/2010-04-01/Accounts/{{data.Account.Sid}}/IncomingPhoneNumbers/*SID*

  • Delete an incoming phone number

    curl -v -X DELETE --user {{data.Account.Sid}}:{{data.Account.AuthToken}} {{data.host}}/2010-04-01/Accounts/{{data.Account.Sid}}/IncomingPhoneNumbers/*SID*
